Network Simulator

In my studies of wired networks, I created a program using Java and JavaFX. The programs are split up into 3 sections. Each section adds new advanced and important features to the network. In the most recent section is the entire network simulator, including the Dijkstras routing algorithm, full CRC and ARQ features, and network layer header implementations.

Below is a demonstration of the dynamic path of the circles, which represent packets of data being transmitted by the network. The red and blue numbers on the wires, that connect the routers, represent the time in seconds that it takes for the respective packet to travel to the next router. The path of the circles to routers are determined by a custom implementation of Dijkstras algorithm. The lines at the bottom of the window represent digital and analog electrical signals. These are transmitted via the physical layer in the network. At the end of the gif a purple packet is sent to the initial computer, this packet represents the message received signal. Nothing about this program is same every run, different text inputs yield different results.

The project was started during the fall 2017 semester, and with only 3 months to complete it there ended up being over 3000 lines of written code. The program needed to be fast, efficient, and clean. Since I wanted it to be object oriented, and since I wanted to learn JavaFX, I decided JavaFX for the front end interface and Java back end would be the best language for this program. The first and second sections of this program are noticably messy in the code. However, by the third assignment I had gone over my functions to make the animation and processing much more efficient.
