Student Information

During my class of concepts of computing languages I was taken out of my Java comfort zone. Given a strict deadline and no experience in C++ I had to be a quick learner. I put this project together in less than a month. Its purpose is to use various types of C++ programming methods to keep track of university affiliate's information.

In the production of this project I had shaken my only Java programming experience. I realized that the best way to get started in a new program is to dive in head first, and learn as you go. Now whenever I have an idea for a new project instead of thinking "how can I make this work in Java" I now think "What language would be best suited for this idea". If I don't know much about the language it's best for I see it as a great opportunity to learn more about it.
